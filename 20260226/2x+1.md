# 从 0 推导单神经元的梯度下降全过程

------

# 一、问题定义

我们假设真实规律是：

```
y = 2x + 1
```

但机器不知道这个公式。

我们希望通过训练，让模型自动学出：

```
w ≈ 2
b ≈ 1
```

------

# 二、模型结构（一个神经元）

我们定义最简单的模型：

```
y_pred = w * x + b
```

只有两个参数：

- w：权重
- b：偏置

这就是一个**单神经元模型**。

------

# 三、损失函数

为了衡量预测值和真实值的差距，我们定义损失函数：

```
L = (y_pred - y_true)^2
```

这个叫做：

- 均方误差（MSE）
- 平方损失

为什么用平方？

- 避免正负抵消
- 误差越大惩罚越严重

------

# 四、训练的核心目标

我们要做的是：

```
不断调整 w 和 b
让 L 变小
```

怎么调整？

靠导数（梯度）。

------

# 五、链式求导完整推导

我们现在推导：

```
dw = ∂L/∂w
db = ∂L/∂b
```

------

## 1️⃣ 定义中间变量

设：

```
e = y_pred - y_true
```

则：

```
L = e^2
```

而：

```
y_pred = wx + b
```

所以：

```
e = (wx + b) - y_true
```

注意依赖关系：

```
w → y_pred → e → L
b → y_pred → e → L
```

这就是链式法则的路径。

------

## 2️⃣ 推导 db

我们要求：

```
∂L/∂b
```

根据链式法则：

```
∂L/∂b = (∂L/∂e) × (∂e/∂b)
```

------

### 第一步：求 ∂L/∂e

因为：

```
L = e^2
```

平方的导数：

```
d(e^2)/de = 2e
```

所以：

```
∂L/∂e = 2e
```

------

### 第二步：求 ∂e/∂b

```
e = wx + b - y_true
```

对 b 求导：

- wx 是常数 → 0
- b 的导数 → 1
- y_true 是常数 → 0

所以：

```
∂e/∂b = 1
```

------

### 合并

```
∂L/∂b = 2e × 1
       = 2e
```

换回 e：

```
db = 2 (y_pred - y_true)
```

------

## 3️⃣ 推导 dw

同样使用链式法则：

```
∂L/∂w = (∂L/∂e) × (∂e/∂w)
```

------

### 第一步：∂L/∂e

已经算过：

```
∂L/∂e = 2e
```

------

### 第二步：∂e/∂w

```
e = wx + b - y_true
```

对 w 求导：

- wx 对 w 求导 → x
- b 常数 → 0
- y_true 常数 → 0

所以：

```
∂e/∂w = x
```

------

### 合并

```
∂L/∂w = 2e × x
       = 2 (y_pred - y_true) x
```

------

# 六、最终梯度公式

```
dw = 2 (y_pred - y_true) x
db = 2 (y_pred - y_true)
```

------

# 七、训练过程完整代码（带打印）

```
w = 0.0
b = 0.0
lr = 0.01

for i in range(20):
    x = 3
    y_true = 7   # 2*3 + 1

    # 1️⃣ 前向计算
    y_pred = w * x + b

    # 2️⃣ 计算损失
    loss = (y_pred - y_true) ** 2

    # 3️⃣ 计算梯度
    dw = 2 * (y_pred - y_true) * x
    db = 2 * (y_pred - y_true)

    # 4️⃣ 更新参数
    w = w - lr * dw
    b = b - lr * db

    print(f"第{i}次:")
    print(f"  y_pred={y_pred:.4f}")
    print(f"  loss={loss:.4f}")
    print(f"  w={w:.4f}, b={b:.4f}")
    print("------------")

print("最终结果:", w, b)
```

------

# 八、训练过程的四个核心步骤

所有神经网络训练都遵循：

```
1. 前向计算（预测）
2. 计算损失
3. 计算梯度
4. 更新参数
```

简称：

```
Forward → Loss → Backward → Update
```

------

# 九、学习率的作用

参数更新公式：

```
w = w - lr * dw
b = b - lr * db
```

学习率（lr）决定步长：

- 太大 → 会震荡
- 太小 → 收敛慢
- 合适 → 稳定下降

------

# 十、本质总结

这个例子告诉我们：

- 神经网络本质是函数
- 训练本质是最小化损失
- 梯度告诉我们参数该往哪走
- 更新参数就是“走一步”

复杂网络只是：

- 多层嵌套
- 多维矩阵
- 多参数版本

核心逻辑完全一样。

------

# 十一、一句话总结

神经网络训练就是：

```
算误差
看误差对参数的影响
往误差变小的方向走
重复很多次
```
